# 住宅予算診断インタラクティブ動画・仕様書
**Project Name:** UriHome Interactive Budget Simulator

## 1. プロジェクト概要
### 1.1 目的
住宅購入検討ユーザーに対し、動画を見ながら手元のワークシートを埋める能動的な体験を提供し、信頼構築と相談への誘導を行う。

## 2. UI/UX仕様
### 2.1 画面レイアウト
* **上部動画エリア:** 画面上部（約30%）に固定表示。スクロールしても常に見える。
* **下部ワークシート:** 動画の下はスクロール可能な入力フォーム（手書きノート風デザイン）。

### 2.2 インタラクション（スポットライト）
* **フォーカス機能:** 動画が指定時間で一時停止した際、入力すべきフォーム以外をグレーアウト（暗く）して注目させる。
* **進行制御:** 入力が完了するまで「次へ」ボタンは非表示。完了してボタンを押すと、グレーアウト解除→動画再開→次の項目へ自動スクロール。

### 2.3 モバイル対応
* **キーボード対策:** 入力時、フォームが隠れないよう自動スクロール調整（scrollIntoView）。
* **開始ボタン:** 初回は「スタート」ボタンを表示し、タップで音あり再生を開始。

## 3. 機能要件
### 3.1 データ保存
* 入力内容はLocalStorageに即時保存し、リロードしても中断箇所から再開可能にする。

### 3.2 予算計算ロジック（簡易版）
* 入力変数: 年収(A), 家賃(B), 貯金(C), 予備費(D)
* 安全月支払い額 = (B + C) - D
* これを逆算して「総予算」を表示する。

### 3.3 結果送信
* 完了時、LIFFのsendMessagesを使い、診断結果（理想・年収・算出された予算）をLINEトークにテキスト送信する。
